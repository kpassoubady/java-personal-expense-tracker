Personal Expense Tracker

A comprehensive solution for managing your personal finances

Key Features

Dashboard Analytics

Comprehensive dashboard with expense summaries, monthly trends, and category breakdowns with interactive charts.

Category Management

Organize expenses with customizable categories, including color coding and icons for easy identification.

Expense Tracking

Record and track expenses with detailed information including date, amount, category, and descriptions.

Advanced Search

Filter and search expenses by date range, category, amount, and description with powerful search capabilities.

Responsive Design

Fully responsive interface that works seamlessly on desktop, tablet, and mobile devices.

REST API

Complete RESTful API for integration with external applications and advanced analytics.

Technology Stack

Backend

  • Java 21: Modern Java runtime with latest features
  • Spring Boot 3.2.3: Application framework and dependency injection
  • Spring Data JPA: Data persistence and repository pattern
  • Hibernate: Object-relational mapping (ORM)
  • H2 Database: In-memory database for development
  • Maven: Build automation and dependency management

Frontend

  • Thymeleaf: Server-side template engine
  • Bootstrap 5: Responsive CSS framework
  • Font Awesome: Icons and visual elements
  • Chart.js: Interactive data visualization
  • HTML5 & CSS3: Modern web standards
  • JavaScript: Dynamic user interactions

RESTful API Endpoints

Category API
  • GET /api/categories - List all categories
  • GET /api/categories/{id} - Get category by ID
  • POST /api/categories - Create category
  • PUT /api/categories/{id} - Update category
  • DELETE /api/categories/{id} - Delete category
  • GET /api/categories/search - Search categories
  • GET /api/categories/statistics - Category stats
Expense API
  • GET /api/expenses - List all expenses
  • GET /api/expenses/{id} - Get expense by ID
  • POST /api/expenses - Create expense
  • PUT /api/expenses/{id} - Update expense
  • DELETE /api/expenses/{id} - Delete expense
  • GET /api/expenses/search - Advanced search
  • GET /api/expenses/analytics - Analytics data
  • GET /api/expenses/category/{id} - By category
